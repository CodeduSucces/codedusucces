Foire aux questions (FAQ)

  • Comment puis-je suivre les progrès de mon enfant ?

    Vous pouvez suivre les progrès de votre enfant en vous connectant régulièrement à l'espace parent. Vous y trouverez les notes, les bulletins scolaires et les commentaires des enseignants.

  • Comment signaler une absence ?

    Pour signaler une absence, vous pouvez utiliser le formulaire en ligne disponible dans l'espace parent ou contacter directement le secrétariat de l'école par téléphone.

  • Comment prendre rendez-vous avec un enseignant ?

    Vous pouvez demander un rendez-vous avec un enseignant via la messagerie de l'espace parent ou en contactant le secrétariat de l'école qui organisera la rencontre.
